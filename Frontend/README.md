# Data Collection Frontend

React + Vite + Tailwind dashboard for viewing collected data.

## Quick Start

```bash
npm install
npm run dev
```

Opens at http://localhost:5173

## Features
- Password-protected login (6-hour auto-lock)
- Sessions dashboard with search & filters
- Interactive data visualization
- Export to JSON

## Login
Default password: admin123 (change in backend)

See full documentation in project root.
